

# 📚 Interactive FlashCard App

An interactive FlashCard application built using **HTML, CSS, and JavaScript** that helps users create, manage, and study flashcards efficiently. The application stores all flashcards using the browser's Local Storage, allowing users to continue learning even after refreshing the page.

---

## ✨ Features

- ➕ Create new flashcards
- 📖 Study mode with card flip animation
- ⬅️ Previous & Next navigation
- 🔀 Shuffle flashcards
- 🗑️ Delete flashcards
- 📊 Progress tracking
- 💾 Local Storage support
- 📱 Responsive UI
- 🎨 Modern Glassmorphism Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

---

## 💡 How It Works

The application allows users to:

- Create flashcards with a question and answer.
- Store flashcards in the browser using Local Storage.
- Review flashcards in Study Mode.
- Flip cards to reveal answers.
- Shuffle cards randomly.
- Navigate using Previous and Next buttons.
